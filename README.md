# AI Creator Production Portfolio

Адаптивный сайт-портфолио для AI Creator Production: AI-видео, реклама, персонажи, Reels, музыкальные эксперименты, промпты и ComfyUI workflow.

## Что внутри

- `index.html` — главная страница портфолио.
- `styles.css` — визуальная система и адаптивная сетка.
- `script.js` — карточки проектов и модальные описания кейсов.
- `portfolio.config.js` — тексты, контакты и список проектов.
- `assets/cartoon-example/` — видео и poster главного кейса с мультфильмом.
- `case-studies/new-case-template.md` — шаблон для нового кейса.

## Как редактировать

1. Откройте `portfolio.config.js`.
2. Замените `owner`, контакты и проекты на свои данные.
3. Для каждого проекта заполните название, направление, краткое описание, результат и production pipeline.
4. Для подробных кейсов создавайте отдельные Markdown-файлы в папке `case-studies`.
5. Проверьте ссылку Telegram в контактном блоке.

## Что показать клиенту первым

- Флагманский кейс с мультфильмом: `Авторский мультфильм`.
- Рекламный формат: `Рекламный концепт продукта`.
- Систему коротких сценариев: `Система сценариев для коротких видео`.
- Процесс работы с несколькими нейросетями.
- Музыкальный визуальный пакет.

## Публикация на GitHub Pages

1. Создайте публичный репозиторий `ai-creator-portfolio`.
2. Загрузите файлы сайта в корень репозитория.
3. Откройте `Settings` -> `Pages`.
4. В `Build and deployment` выберите `Deploy from a branch`.
5. Укажите ветку `main` и папку `/ (root)`.
6. Сохраните настройки.
